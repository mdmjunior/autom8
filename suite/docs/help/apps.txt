AutoM8 help: autom8 apps

Status: available
Desde: 0.2.0

Instala e remove aplicativos por catálogo local atualizável online.

Detalhes:
- O módulo Apps usa catálogo local consolidado em suite/catalog/apps.json.
- O catálogo fonte é separado por categoria em suite/catalog/apps/.
- O catálogo consolidado é gerado por scripts/build-apps-catalog.sh.
- A CLI lê o catálogo consolidado para listar, buscar, detalhar, instalar e remover apps.
- Apps com status available podem ser instalados/removidos pelos repositórios da distro.
- Apps com status advanced aparecem no catálogo, mas não são instalados automaticamente nesta versão.
- Toda instalação e remoção real exige confirmação explícita.
- O modo --dry-run mostra o comando previsto sem executar alteração real.
- Categorias iniciais: desenvolvimento, produtividade, containers, sistema, rede, mídia, design, games, escritório e usuário local.

Exemplos:
  autom8 apps
  autom8 apps categories
  autom8 apps list
  autom8 apps list --category desenvolvimento
  autom8 apps list --category escritorio
  autom8 apps search docker
  autom8 apps show git
  autom8 --dry-run apps install git
  autom8 apps install htop
  autom8 --dry-run apps install-many git htop jq
  autom8 apps install-many git htop jq
  autom8 --dry-run apps remove htop
  autom8 apps remove htop
  autom8 --dry-run apps remove-many git htop
  autom8 apps remove-many git htop
  autom8 apps update-catalog
