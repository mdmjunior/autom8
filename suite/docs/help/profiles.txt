AutoM8 help: autom8 profiles

Status: available
Desde: 0.2.0

Lista, detalha, instala e remove perfis baseados no catálogo de apps.

Detalhes:
- O catálogo atual possui seis perfis funcionais.
- Os perfis utilizam apps disponíveis no catálogo consolidado.
- Instalação e remoção reutilizam os fluxos do módulo Apps e exigem confirmação explícita.
- Os comandos disponíveis são list, show, install e remove.

Exemplos:
  autom8 profiles list
  autom8 profiles show dev-essential
  autom8 profiles install dev-essential
  autom8 profiles remove dev-essential
