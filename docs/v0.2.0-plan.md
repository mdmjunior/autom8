# AutoM8 v0.2.0 - Planejamento

## Objetivo

Implementar o módulo de Apps e Perfis YAML.

## Escopo inicial

- Ler catálogos YAML por distro.
- Listar apps disponíveis.
- Filtrar apps por categoria.
- Instalar apps nativos por gerenciador de pacotes.
- Remover apps nativos.
- Detectar Flatpak.
- Habilitar Flathub quando necessário.
- Usar Snap somente quando já estiver instalado.
- Ler perfis YAML.
- Permitir modo mínimo seguro.
- Permitir modo completo.
- Permitir marcar/desmarcar apps antes de aplicar perfil.

## Fora do escopo inicial

- Interface web.
- Gerenciamento remoto.
- Banco de dados.
- Instalação automática de snapd.
- Rollback completo.
