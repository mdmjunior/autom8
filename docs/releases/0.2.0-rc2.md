# AutoM8 0.2.0-rc2

Release candidate 2 da versão 0.2.0 do AutoM8.

Esta versão corrige o fluxo de instalação observado no teste do RC1 em VM Ubuntu Desktop.

## Correção principal

O RC1 apresentou falha pós-instalação:

    Executável não encontrado ou sem permissão: /opt/autom8/bin/autom8

O instalador foi reforçado para:

- validar se o arquivo baixado é um tar.gz válido;
- inspecionar o conteúdo do pacote antes da instalação;
- detectar automaticamente a raiz real do pacote;
- localizar `bin/autom8` mesmo se o pacote vier com uma pasta raiz inesperada;
- normalizar a instalação para `/opt/autom8`;
- exibir diagnóstico útil quando o pacote baixado não contém `bin/autom8`.

## Mantido do RC1

- Catálogo de apps separado por categorias.
- Catálogo consolidado em `suite/catalog/apps.json`.
- Instalação e remoção individual, múltipla e por categoria.
- Bloqueio de apps `advanced` e `planned`.
- Perfis baseados no catálogo.
- Smoke test para Ubuntu Desktop.
- Validação pós-instalação de RC.

## Teste recomendado

Em VM Ubuntu Desktop descartável:

    sudo apt update
    sudo apt install -y curl git jq

    curl -fsSL https://autom8.oslabs.com.br/install.sh -o /tmp/autom8-install.sh

    AUTOM8_PACKAGE_URL=https://github.com/mdmjunior/autom8/releases/download/v0.2.0-rc2/autom8-0.2.0-rc2.tar.gz bash /tmp/autom8-install.sh

Depois:

    export PATH="/opt/autom8/bin:$PATH"

    autom8 --version
    autom8 doctor
    autom8 apps categories
    autom8 profiles list
    autom8 --dry-run apps install-category sistema
    autom8 --dry-run profiles install dev-essential

## Aviso

Esta ainda é uma prerelease. Use apenas em VM descartável ou ambiente de teste.
