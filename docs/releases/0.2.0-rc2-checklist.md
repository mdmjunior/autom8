# Checklist AutoM8 0.2.0-rc2

## Objetivo

Validar a correção do instalador após falha do RC1 em VM Ubuntu Desktop.

## Antes de publicar

    git checkout feature/apps-v0.2
    git pull --ff-only origin feature/apps-v0.2
    git status

Working tree deve estar limpo antes da publicação.

## Validações locais

    bash -n installer/install.sh
    bash -n site/public/install.sh
    bash -n scripts/release-rc.sh
    bash -n scripts/smoke-ubuntu-desktop.sh
    bash -n scripts/verify-installed-rc.sh

    ./scripts/verify-installer.sh
    ./scripts/smoke-ubuntu-desktop.sh
    ./scripts/build-site.sh
    ./scripts/package.sh

## Publicar RC2

    ./scripts/release-rc.sh

Como `suite/VERSION` estará em `0.2.0-rc2`, o script deve criar:

    v0.2.0-rc2

Assets esperados:

    autom8-0.2.0-rc2.tar.gz
    autom8-rc-latest.tar.gz

## Testar instalador com pacote RC2

Na VM Ubuntu Desktop descartável:

    sudo rm -rf /opt/autom8
    sudo apt update
    sudo apt install -y curl git jq

    curl -fsSL https://autom8.oslabs.com.br/install.sh -o /tmp/autom8-install.sh

    AUTOM8_PACKAGE_URL=https://github.com/mdmjunior/autom8/releases/download/v0.2.0-rc2/autom8-0.2.0-rc2.tar.gz bash /tmp/autom8-install.sh

## Validar instalação

    export PATH="/opt/autom8/bin:$PATH"

    autom8 --version
    autom8 doctor
    autom8 apps categories
    autom8 apps list --category sistema
    autom8 profiles list
    autom8 --dry-run apps install-category sistema
    autom8 --dry-run profiles install dev-essential

## Verificação pós-instalação

Pelo clone da branch:

    AUTOM8_BIN=/opt/autom8/bin/autom8 AUTOM8_EXPECTED_VERSION=0.2.0-rc2 ./scripts/verify-installed-rc.sh

## Critério de aprovação

O RC2 pode ser aprovado se:

- instalação cria `/opt/autom8/bin/autom8`;
- `autom8 --version` mostra `0.2.0-rc2`;
- doctor executa;
- apps funcionam em dry-run;
- profiles funcionam em dry-run;
- app advanced continua bloqueado;
- nenhuma falha crítica ocorre na VM.

## Smoke test Fedora Workstation

Além do Ubuntu Desktop, valide também em Fedora Workstation:

    sudo dnf install -y git curl jq tar gzip

    git clone -b feature/apps-v0.2 https://github.com/mdmjunior/autom8.git
    cd autom8

    ./scripts/smoke-fedora-workstation.sh

Modo estrito opcional:

    AUTOM8_FEDORA_STRICT=1 ./scripts/smoke-fedora-workstation.sh

## Teste do instalador RC2 em Fedora

    sudo rm -rf /opt/autom8
    sudo dnf install -y curl git jq tar gzip

    curl -fsSL https://autom8.oslabs.com.br/install.sh -o /tmp/autom8-install.sh

    AUTOM8_PACKAGE_URL=https://github.com/mdmjunior/autom8/releases/download/v0.2.0-rc2/autom8-0.2.0-rc2.tar.gz bash /tmp/autom8-install.sh

Depois:

    export PATH="/opt/autom8/bin:$PATH"

    autom8 --version
    autom8 doctor
    autom8 apps categories
    autom8 profiles list
    autom8 --dry-run apps install git
